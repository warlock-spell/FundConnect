This web application, FundConnect, is created by Daksh Gaur.

For more information, you can visit www.daksh.fyi.

To contact Daksh Gaur, you can send an email to hi@daksh.fyi.

The source code for this app can be found on GitHub at github.com/warlock-spell/FundConnect.

This app is built with Django and Tailwind CSS. Contributions are welcome and appreciated. Feel free to contribute to the development of this app.

Setting Up the Project Locally

  1. Clone the repository using the following command: git clone https://github.com/warlock-spell/FundConnect.git
  2. Navigate to the project directory: cd FundConnect
  3. Create a virtual environment (optional but recommended): python3 -m venv env
  4. Activate the virtual environment: source env/bin/activate
  5. Install the project dependencies: pip install -r requirements.txt
  6. Apply database migrations: python manage.py migrate
  7. Run the development server: python manage.py runserver
  8. Open your web browser and visit http://localhost:8000 to access the application.
